# 建立您的第一個服務人員

瞭解如何設定您的第一個Adobe Brand Concierge ，協助客戶尋找正確的Adobe產品、服務或體驗。 您會學習如何為門房命名、新增有用的內容、選擇回應方式，以及讓聲音與您的品牌相配。 一旦準備就緒後，您就可以加以測試，並瞭解它在桌上型電腦和行動裝置上的運作方式。

## 這部影片是給誰看的？

* 數位Experience Manager
* 行銷技術人員
* 客戶體驗Designer

## 視訊內容

* 如何設定Adobe Brand Concierge
* 如何連線內容來源以取得準確回應
* 如何自訂品牌語調並測試體驗
